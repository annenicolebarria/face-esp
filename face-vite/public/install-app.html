2

3

Add it to the device

Install it so the app launches in its own window like a normal app.

Platform Guides

Browser-specific instructions

Chrome or Edge on Desktop

  • Open the PWA app page.
  • Look for the install icon in the address bar.
  • If there is no icon, open the browser menu and choose Install app or Apps.

Android Chrome

  • Open the PWA app page in Chrome.
  • Tap the three-dot menu.
  • Choose Install app or Add to Home screen.

iPhone Safari

  • Open the PWA app page in Safari.
  • Tap the Share button.
  • Choose Add to Home Screen, then confirm.
